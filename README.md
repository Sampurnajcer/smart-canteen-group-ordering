# Smart Canteen System 🍽️

A web-based Smart Canteen System designed to simplify and modernize food ordering in institutions. It allows users to place orders online, reducing waiting time and improving overall efficiency.

##  Features
- Online food ordering system  
- Unique Order ID for each order  
- Group ordering with single Group Order ID  
- Menu browsing and add-to-cart functionality  
- Order management for canteen staff  
- Pay-at-counter model  

## Tech Stack
- Frontend: React.js, JavaScript, Material UI  
- Backend: Python, Flask  
- Database: SQLite  

##  Description
The system replaces traditional manual ordering with a digital platform, helping reduce long queues, avoid miscommunication, and improve order accuracy. The group ordering feature allows multiple users to combine orders into a single request, making it easier for both users and staff during peak hours.

## Future Scope
- Online payment integration  
- Real-time order tracking  
- Notifications and alerts  
- Inventory management
